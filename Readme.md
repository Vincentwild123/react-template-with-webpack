### 📄Description

**A template to create React App , integrate `Webpack`、`React-router`、`React-redux`、`React Hooks`、`Axios`**

### 🔑Usage

**1. clone the repo**

```shell
$ git clone
```

**2. get into the folder**

```shell
$ cd folder-name
```

**3. install the dependences**

```shell
$ yarn
```

**4. run**

```shell
$ npm start
```
